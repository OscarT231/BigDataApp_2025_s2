# Descripción 
Este es repositorio que  contendrá ejercicios de matemáticas provenientes de  talleres, simulacros, guías y pruebas liberadas para la preparación de pruebas de estado para grado once. 
El objetivo del proyecto es contruir una plataforma que reúne, limpia, organiza y clasifica los ejercicios mediante web scraping y NLP
